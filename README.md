# Bluestock Mutual Fund Analytics System

## Project Overview

The Bluestock Mutual Fund Analytics System is a comprehensive financial analytics project developed using Python, SQLite, SQL, and Power BI. The project automates mutual fund data processing, performs performance analysis, evaluates investment risk, and provides interactive dashboards for informed investment decisions.

## Features

- Automated ETL Pipeline
- Data Cleaning & Preprocessing
- SQLite Database Integration
- Exploratory Data Analysis (EDA)
- CAGR, Sharpe Ratio, Beta
- Historical VaR & CVaR
- Rolling Sharpe Ratio
- Investor Cohort Analysis
- SIP Continuity Analysis
- Sector HHI Analysis
- Mutual Fund Recommendation System
- Interactive Power BI Dashboard

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- SQLite
- SQL
- Power BI
